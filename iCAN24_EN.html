What is Phox

Phox is a device with camera to monitor mail/delivery boxes by notifing the opening of boxes.

First, you may look deeply on the device.

Hardware Structure

Main board for the Phox is completely order made desing based on commercially available ESP32-S3 board with camera.

Main processor is ESP32-S3 and using OV5640 as a camera module.

The disintegurated image is as shown on below.

The board uses 4 layer circuit enabling high density of electronic part, thus the device is pretty small and thin.

This effort makes it possible downsizing the device by 50% compared to the privious version.

We also donsized open/close sensor unit by 30mm x 20mm.

Software System

We integurated AWS (Amazon Web Service) to this device as a cloud technology.

The system has 3 features.

  • API that store Base64 encoded image to AWS S3 as a jpg iamge.
  • Lambda that runs image recogntion for stored AWS S3 image.
  • Lambda that send image and the result of the image recogntion to registered e-mail address.

The overall system of the AWS is as shown below.

The mail send to users have 3 elements: 1. image inside mail/delivery box, 2. result of iamge recogntion, 3. current battery volatage.

Image recogntion is not introduced to the current Phox system. Though, the image recogntion program with XGBoost marks 95% accuracy wtih test envioronment.

This AWS system enable instant notification of image inside mail/delivery box.
